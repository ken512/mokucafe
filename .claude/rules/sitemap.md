## 画面遷移フロー

```
[ホーム /]
  ├── ログイン → /login
  ├── 新規登録 → /signup
  ├── 募集詳細 → /posts/[id]
  │     └── 参加申請（要ログイン）
  └── 募集を投稿する → /posts/new（要ログイン）

[ログイン /login]
  └── 新規登録リンク → /signup

[新規登録 /signup]
  └── ログインリンク → /login
```

## 注意事項

- `/register` は使わない。新規登録は必ず `/signup`
- ゲストユーザーは全ページを閲覧可能（投稿・申請はログイン必要）
- MVPスコープ：募集投稿〜参加申請の流れのみ。マッチング機能は対象外
